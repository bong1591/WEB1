HTML이란 무엇인가?

Hypertext Markup language (HTML) is the standard markup language for creating web pages and web applications. Web browsers recrive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page sementifically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interaactive forms, may be embeded into the rendered page. It provides a means to create structured documents by denoting structurarl semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
